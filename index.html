SVG Plot Module Extension

Information

The SVG Plot Module Extension consists of multiple modules, each of which performs a specific task within the parsing of ‘plot’ elements.
The SVG Plot modules are loaded through the SVGModule object. This object, created by the SVGPlotModule.js file, provides an interface which is loosely derived from the CommonJS Modules 1.0 specification.
Refer to the Wiki for more project information or the Plots Page for more information on the ‘plot’ element.

Usage

The SVGPlotModule.js file needs to be loaded into the SVGDocument with the ‘script’ element: 

<svg xmlns="http://www.w3.org/2000/svg"
     xmlns:xlink="http://www.w3.org/1999/xlink"
     …>
  <script xlink:href="http://rmcuenen.github.io/svgplot/SVGPlotModule.js" />
  <plot … />
</svg>

Note that this will only create the SVGModule object through which the required modules will be loaded.
How the SVGModule object resolves the modules depends on the following configuration:

  • By default the module files are assumed to be relative to the address of the SVG document.
  • When the ‘script’ element has the ‘id’ attribute set to "svgplot-loader" the module files are assumed to be relative to the address of the SVGPlotModule.js file.
  • If the "svgplot-loader" element also has a ‘base’ attribute set. The module files are assumed to be relative to that attribute's value.

Now the SVGPlotHandler module (and optionally the DocumentScaler module) can be loaded.
To load (or require) modules (through the SVGModule object) add the following ‘script’ element: 

<svg xmlns="http://www.w3.org/2000/svg"
     xmlns:xlink="http://www.w3.org/1999/xlink"
     …>
  <script xlink:href="http://rmcuenen.github.io/svgplot/SVGPlotModule.js"
          id="svgplot-loader" base="http://rmcuenen.github.io/svgplot/modules" />
  <script>
  <![CDATA[
    SVGModule.require(["DocumentScaler", "SVGPlotHandler"],
      function(DocumentScaler) {
        DocumentScaler.scaleTo(document.getElementById("test-plot"));
      }
    );
  ]]>
  </script>
  <plot id="test-plot" … />
</svg>

Testing

The SVG Plot Module Extension is tested with TestNG and the use of the WebDriver API of Selenium 2.0.
Tests are divided into two groups: all and tests resulting in an alert (The OperaDriver has no support for JavaScript alert/popup dialogues).
The SVG Plot Module Extension is tested and guaranteed to run on the following browsers:

  • All tests passed on Google Chrome version 31.0.1650.63
  • All tests passed on Mozilla Firefox version 26.0
  • All tests, excluding the alert group, passed on Opera version 12.16
